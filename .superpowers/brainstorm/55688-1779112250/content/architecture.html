Arsitektur Aplikasi

Teknologi dan struktur yang akan digunakan.

Frontend (Vite + React)
  • Tailwind CSS: Styling minimalist.
  • Framer Motion: Animasi smooth.
  • React Hook Form: Handle RSVP.
  • Lucide React: Icon set.
  • Html5-qrcode: Library scan barcode di CMS.
Backend (Supabase)
  • PostgreSQL: Database utama.
  • Supabase Auth: Login admin untuk CMS.
  • Edge Functions: (Optional) generate QR if needed.

Kelebihan Pendekatan Ini

  • Deployment gratis (Vercel/Netlify).
  • Database real-time (update souvenir langsung kelihatan).
  • Performa kencang (Vite).
